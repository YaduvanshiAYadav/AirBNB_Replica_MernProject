# Airbnb Clone (MERN)

Full-stack property rental platform.

## Features
- Authentication
- Create listing
- Edit/Delete listing
- Cloud image upload
- Booking flow
- Review/Share Experience

## Tech Stack

Express
Node
Cloudinary
